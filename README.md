# 🧮 Calculadora Básica en Java

Una calculadora simple desarrollada en Java, pensada especialmente para ejecutarse en Eclipse. Este proyecto está diseñado para ser claro, funcional y fácil de entender, ideal para aprendizaje o demostraciones.

---

## ✨ Características

* Operaciones básicas:

  * ➕ Suma
  * ➖ Resta
  * ✖️ Multiplicación
  * ➗ División
* Interfaz por consola (simple y rápida)
* Código limpio y fácil de mantener
* Manejo básico de errores (como división por cero)

---

## 🚀 Cómo ejecutar el proyecto

1. Abre Eclipse.
2. Crea un nuevo proyecto Java.
3. Añade una nueva clase (por ejemplo: `Calculadora.java`).
4. Copia y pega el código en la clase.
5. Ejecuta el programa como **Java Application**.

---

## 💻 Ejemplo de uso

```
Selecciona una operación:
1. Sumar
2. Restar
3. Multiplicar
4. Dividir

Opción: 1
Introduce el primer número: 5
Introduce el segundo número: 3

Resultado: 8
```

---

## ⚠️ Consideraciones

* La división entre cero no está permitida.
* Solo se aceptan números válidos (puedes mejorar esto con validaciones extra).
* Este proyecto es una base: puedes ampliarlo con interfaz gráfica (Swing o JavaFX).
